## 📌 Sobre o projeto

O **Generate AI** é um projeto acadêmico que utiliza a **API do GroqCloud** para gerar conteúdo automaticamente a partir de uma ideia fornecida pelo usuário.

A aplicação conta com uma interface simples e intuitiva que permite ao usuário escolher diferentes tipos de saída, como postagens para redes sociais, títulos, resumos e reescritas de textos — demonstrando, na prática, o uso de inteligência artificial em aplicações web.

> Desenvolvido com foco em consumo de APIs, desenvolvimento front-end e experiência do usuário.

---

## ✨ Funcionalidades

- 💬 **Postagens para redes sociais** — gera textos prontos para publicar
- 🏷️ **Geração de títulos** — cria títulos criativos a partir de uma ideia
- 📝 **Resumos automáticos** — condensa textos longos em pontos principais
- ✏️ **Reescrita de textos** — reformula conteúdos com nova perspectiva
- ⚡ **Respostas rápidas** — powered by GroqCloud (LLMs de alta velocidade)

## 🚀 Como rodar o projeto

### Pré-requisitos

- Navegador moderno (Chrome, Firefox, Edge)
- Chave de API válida do [GroqCloud](https://console.groq.com/)

### Passos

```bash
# Clone o repositório
git clone https://github.com/GiovanniBolsoni/Soltaacarta.git

# Acesse a pasta do projeto
cd Soltaacarta
```

Abra o arquivo `index.html` no seu navegador — ou use uma extensão como **Live Server** no VS Code.

> ⚠️ Antes de usar, insira sua chave de API do GroqCloud no local indicado no código (variável de configuração ou campo na interface).

---

## 🔑 Configuração da API

1. Acesse [console.groq.com](https://console.groq.com/) e crie uma conta gratuita
2. Gere uma **API Key** no painel
3. Insira a chave no projeto conforme indicado no arquivo de configuração

```.env
GROQ_API_KEY="sua_chave_aqui"
```

> 🔒 Nunca compartilhe sua chave de API publicamente nem a envie para o repositório.
